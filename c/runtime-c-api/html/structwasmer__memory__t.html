#include <wasmer.h>

Detailed Description

Opaque pointer to a wasmer_vm::Memory value in Rust.

A wasmer_vm::Memory represents a WebAssembly memory. It is possible to create one with wasmer_memory_new() and pass it as imports of an instance, or to read it from exports of an instance with wasmer_export_to_memory().
