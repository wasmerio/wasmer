Struct ThreadConditionsHandle

pub struct ThreadConditionsHandle {
    inner: Weak<NotifyMap>,
}
Expand description

A weak handle to a ThreadConditions instance, which does not prolong its lifetime.

Internally holds a std::sync::Weak pointer.
