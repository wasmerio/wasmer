Struct Exception

pub struct Exception(pub(crate) BackendException);
Expand description

A WebAssembly exception instance.

An exception is an internal construct in WebAssembly that represents a runtime object that can be thrown. A WebAssembly exception consists of an exception tag and its runtime arguments.

Spec: https://github.com/WebAssembly/exception-handling/blob/main/proposals/exception-handling/Exceptions.md#exceptions
