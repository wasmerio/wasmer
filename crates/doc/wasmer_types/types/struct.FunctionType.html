Struct FunctionType 

Source 
pub struct FunctionType {
    params: Box<[Type]>,
    results: Box<[Type]>,
}
Expand description

The signature of a function that is either implemented in a Wasm module or exposed to Wasm by the host.

WebAssembly functions can have 0 or more parameters and results.

Fields§
