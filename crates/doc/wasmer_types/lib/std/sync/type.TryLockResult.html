Type Definition wasmer_types::lib::std::sync::TryLockResult1.0.0[−][src]

type TryLockResult<Guard> = Result<Guard, TryLockError<Guard>>;

A type alias for the result of a nonblocking locking method.

For more information, see LockResult. A TryLockResult doesn’t necessarily hold the associated guard in the Err type as the lock may not have been acquired for other reasons.
