pub struct BoxedSlice<K, V> where
    K: EntityRef,  {
    elems: Box<[V]>,
    unused: PhantomData<K>,
}
Expand description

A slice mapping K -> V allocating dense entity references.

The BoxedSlice data structure uses the dense index space to implement a map with a boxed slice.
