pub enum Value<T> {
    I32(i32),
    I64(i64),
    F32(f32),
    F64(f64),
    ExternRef(ExternRef),
    FuncRef(Option<T>),
    V128(u128),
}
Expand description

Possible runtime values that a WebAssembly module can either consume or produce.

Variants

I32(i32)

Tuple Fields

0: i32

A 32-bit integer.

In Wasm integers are sign-agnostic, i.e. this can either be signed or unsigned.

I64(i64)

Tuple Fields

0: i64

A 64-bit integer.

In Wasm integers are sign-agnostic, i.e. this can either be signed or unsigned.

F32(f32)

Tuple Fields

0: f32

A 32-bit float.

F64(f64)

Tuple Fields

0: f64

A 64-bit float.

ExternRef(ExternRef)

Tuple Fields

0: ExternRef

An externref value which can hold opaque data to the wasm instance itself.

Note that this is a nullable value as well.

FuncRef(Option<T>)

Tuple Fields

0: Option<T>

A first-class reference to a WebAssembly function.

Tuple Fields

0: u128
