Enum GlobalInit

pub enum GlobalInit {
    I32Const(i32),
    I64Const(i64),
    F32Const(f32),

    V128Const(V128),
    GetGlobal(GlobalIndex),

Expand description

Globals are initialized via the const operators or by referring to another import.
