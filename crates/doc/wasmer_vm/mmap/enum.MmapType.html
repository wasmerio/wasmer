Enum MmapType

Source 
pub enum MmapType {
    Private,
    Shared,
}
Expand description

The type of mmap to create

Variants§

§

Private

The memory is private to the process and not shared with other processes.

§

Shared

The memory is shared with other processes. This is only supported on Unix. When the memory is flushed it will update the file data.
