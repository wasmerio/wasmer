wasmer_vm

Module wasmer_vm::instance[−][src]

An Instance contains all the runtime state used by execution of a WebAssembly module (except its callstack and register state). An InstanceRef is a wrapper around Instance that manages how it is allocated and deallocated. An InstanceHandle is a wrapper around an InstanceRef.
