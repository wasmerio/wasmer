Expand description

A Source which can query arbitrary packages on the internet.

ยงImplementation Notes

Unlike other Source implementations, this will need to download a package if it is a PackageSource::Url. Optionally, these downloaded packages can be cached in a local directory.

After a certain period (WebSource::with_retry_period()), the WebSource will re-check the uploaded source to make sure the cached package is still valid. This checking is done using the ETag header, if available.
