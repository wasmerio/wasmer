Enum ArtifactBuildVariant

Source 
pub enum ArtifactBuildVariant {
    Plain(ArtifactBuild),
    Archived(ArtifactBuildFromArchive),
}
Expand description

Artifacts may be created as the result of the compilation of a wasm module, corresponding to ArtifactBuildVariant::Plain, or loaded from an archive, corresponding to ArtifactBuildVariant::Archived.
