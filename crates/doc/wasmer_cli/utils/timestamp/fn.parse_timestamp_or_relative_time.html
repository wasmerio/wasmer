Try to parse the string as a timestamp in a number of well-known formats.

Supported formats,

  • RFC 3339 (2006-01-02T03:04:05-07:00)
  • RFC 2822 (Mon, 02 Jan 2006 03:04:05 MST)
  • Date (2006-01-02)
  • Unix timestamp (1136196245)
  • Relative time (10m / -1h, 1d)
