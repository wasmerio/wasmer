Enum wasmer_config::package::ModuleReference

source · 
pub enum ModuleReference {
    CurrentPackage {
        module: String,
    },
    Dependency {
        dependency: String,
        module: String,
    },
}
Expand description

A reference to a module which may or may not come from another package.

Serialization

A ModuleReference is serialized via its String representation.
