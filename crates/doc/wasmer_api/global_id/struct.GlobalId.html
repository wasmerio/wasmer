pub struct GlobalId {
    kind: NodeKind,
    database_id: u64,
}
Expand description

Global id of backend nodes.

IDs are encoded using the “hashid” scheme, which uses a given alphabet and a salt to encode u64 numbers into a string hash.

Fields§

§kind: NodeKind

The node type of the ID.

§database_id: u64

The database ID of the node.
