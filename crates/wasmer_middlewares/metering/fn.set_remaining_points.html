pub fn set_remaining_points(instance: &Instance, points: u64)

Set the provided remaining points in an Instance.

This can be used in a headless engine after an ahead-of-time compilation as all required state lives in the instance.

Panic

The instance Module must have been processed with the Metering middleware at compile time, otherwise this will panic.
