pub unsafe extern "C" fn lto_codegen_compile_optimized(
    cg: lto_code_gen_t, 
    length: *mut size_t
) -> *mut c_void

Generates code for the optimized merged module into one native object file.

Does not run IR optimizations on the merged module.

Returns a pointer to the generated mach-o/ELF buffer with length set to the buffer size. This buffer is owned by cg and will be freed when lto_codegen_dispose is called or lto_codegen_compile_optimized is called again. Returns null on failure.

Added in LLVM 3.7.
