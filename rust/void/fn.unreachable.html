pub fn unreachable(x: Void) -> !

A safe version of intrinsincs::unreachable.

If this typechecks, anything that causes this to run is unreachable code.

Calling this function in reachable code invokes undefined behavior, but should not be possible unless unsafe was used elsewhere to construct an instance of Void (which is already undefined behavior).
