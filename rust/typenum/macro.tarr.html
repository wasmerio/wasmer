[−][src]Macro typenum::tarr

macro_rules! tarr {
    () => { ... };
    ($n:ty) => { ... };
    ($n:ty,) => { ... };
    ($n:ty, $($tail:ty),+) => { ... };
    ($n:ty, $($tail:ty),+,) => { ... };
}

Create a new type-level arrray. Only usable on Rust 1.13.0 or newer.

There's not a whole lot you can do with it right now.

Example

#[macro_use]
extern crate typenum;
use typenum::consts::*;

type Array = tarr![P3, N4, Z0, P38];
