Entry for an existing key-value pair or a vacant location to insert one.

Variants

Occupied(OccupiedEntry<'a, K, V>)

Existing slot with equivalent key.

Vacant(VacantEntry<'a, K, V>)

Vacant slot (no equivalent key in the map).
